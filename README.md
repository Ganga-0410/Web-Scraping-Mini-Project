# 🌐 Web Scraping Mini Project

## 📖 Overview
Welcome to the Web Scraping Mini Project! This tool is designed to extract valuable data from websites and store it in a structured format. Built with Python, it leverages powerful libraries like BeautifulSoup and Requests to make web scraping a breeze.

## ✨ Features
- **Multi-page scraping**: Extract data from multiple web pages effortlessly.
- **CSV storage**: Save scraped data in a clean and organized CSV format.
- **HTTP handling**: Manage HTTP requests and responses seamlessly.
- **HTML parsing**: Efficiently parse HTML content to get the data you need.

## 🚀 Installation
```bash
pip install requests
pip install bs4
pip install openpyxl
```

## 📦 Dependencies
- Python 3.x
- BeautifulSoup
- Requests
